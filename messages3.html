Kata kampus merdeka ada kendala server, tpi aku tadi nyoba kayak gini

1. Logout
2. Close tab web KM
3. Clear cache dan history.
4. Open dan login di web KM
5. Tambahkan NPSN di kolom asal sma bagian tab data Pribadi, Ex: (00000000) SMAN 1 SURABAYA.
6. Verifikasi NISN berdasarkan NISN
7. Error langsung masuk halaman ajukan validasi dan verifikasi.
8. Isi kolom nya sampai ada icon centang warna ijo kecuali kolom sekolah dan file ijazah. (saran saya untuk kolom teks berupa huruf kapital semua tpi dilihat dulu juga di ijazah nya)
